Introduction

Atom is a text editor created by github. It has a lot of robust features out-the-box, but like many modern text editors, functionality is extended with plugins and themes (called packages in Atom). After a brief overview of the editor itself, this presentation will show you a few ways (with some quick examples) to get started writing Atom packages.

Atom Overview

  • Initially Released February 26 2014
  • Based on Chromium
  • Written in Less and CoffeeScript
  • Atom's shell, Electron, was initally released as Atom Shell, now used in popular applications.

apm publish --rename $new-name
